I've been vibe coding for a while now, and since I prefer working with pure Vanilla CSS, I made a starter template to speed up setting up new projects. I didn't want every project to feel identical, so I built this tool to mix things up when needed. There are plenty of tools out there, but sometimes they don't match my exact workflow. This is the second version of something I built purely for myself, and many parts of it are still experimental. The idea is to build something that I'd use regularly.
